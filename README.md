# Guess Game in Python

Um joguinho de adivinhação desenvolvido em Python.

## Download do executável para Windows

1. Vá até a aba **Actions** deste repositório
2. Clique no workflow mais recente
3. Role até o final e baixe o artefato **Guess-game-windows**
4. Extraia o arquivo ZIP e execute o `main.exe`

## Desenvolvimento

Criado com Python 3.12 e PyInstaller para compilação.
